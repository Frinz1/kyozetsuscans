Full Stack Developer

As a passionate manga enthusiast and web developer, I created this Manga Reader to provide a seamless reading experience for fellow manga lovers. This project combines my love for coding and Japanese comics into a user-friendly platform.

The application is built using modern web technologies and integrates with the MangaDex API to provide a vast library of manga titles. My goal was to create an intuitive interface that makes it easy to discover, read, and keep track of your favorite manga series.

Technologies Used

HTML5 CSS3 JavaScript Node.js Express MangaDex API
